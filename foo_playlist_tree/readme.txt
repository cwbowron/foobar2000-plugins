Playlist Tree - By Christopher Bowron - chris@bowron.us

To install, copy the foo_playlist_tree.dll into your components subdirectory 
in your foobar2000 installation directory, and copy the libmz*.dll files into
your root foobar2000 installation directory.  Failure to do so will result in 
problems.

For more information about playlist tree visit: 
http://wiki.bowron.us/index.php/Playlist_Tree
